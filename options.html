⚠ Importante (leia)

Isto é um impeditivo, não segurança de nível militar. Um usuário avançado pode desligar a extensão pelo about:addons ou pelo Modo de Segurança. Enquanto travado, a extensão já joga o about:addons de volta pra tela de senha, mas para travar de verdade use o arquivo policies.example.json (BlockAboutAddons + DisableSafeMode) — veja o README. E lembre: extensão não assinada instalada por about:debugging some ao fechar o Firefox; leia o README pra deixar permanente.
